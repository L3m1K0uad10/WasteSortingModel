# **AI Waste Sorting System (Great for Sustainability)**

A computer vision system that classifies waste into categories.

**Classes**:
- Plastic
- Metal
- Paper
- Glass
- Organic waste
<br/>

**Objective**:
Build an AI system to assist automated recycling by classifying waste images using CNN models.
<br/>

**What will make it impressive**

Instead of basic classification, It will include:
- **Real-time classification with webcam**
- **3D simulation**
<br/>

**Dataset**
- [TrashNet dataset](https://github.com/garythung/trashnet/blob/master/data/dataset-resized.zip)
- TACO dataset
